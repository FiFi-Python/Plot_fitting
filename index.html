Welcome to Plot Image Fitter’s trchnical documentation!¶

Plot Image Fitter is a university project made by me (Adam Strach) and my friend Radosław Kluczewski. The resulting programme is ment to be able to rotate any plot image to its natural horizzontal position and extract data points from an image. The code can be found on github <https://github.com/FiFi-Python/Plot_fitting>

Note

This project is just a basic version made for university class. Problems may occur while using it and fixes may not come, but you are welcome to try to solve them, or notify us by writing to Adam.strach@student.uj.edu.com
